Welcome to the Kwartz-php home page.

Kwartz-php is a template system which realized the concept 'Separation of Presentaion Logic and Presentation Data'.

Kwartz-php has the following features.

    .* Kwartz-php can separate presentation logic from a template.
    .* Kwartz-php runs very fast.
    .* Kwartz-php can be available in Ruby, PHP and Java.
    .* Kwartz-php doesn't break HTML design at all.
    .* Kwartz-php can handle any type of text file.
    .* Kwartz-php can sanitize automatically.
