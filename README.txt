.$ What is Kwartz and Kwartz-java?

  Kwartz is a template system which realized the concept
  'Independence of Presentation Logic'(IoPL).
  Kwartz-java is an implementation of Kwartz in Java.

  See Kwartz Home Page for details.
    http://www.kuwata-lab.com/kwartz
